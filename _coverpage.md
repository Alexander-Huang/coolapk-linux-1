# Coolapk-Linux

> 第三方archlinux软件源.

* 添加以下内容到/etc/pacman.conf，然后pacman -Syy即可使用

```
[coolapk-linux]
SigLevel = Optional TrustAll
Server = http://alexander-huang.gitee.io/coolapk-linux/$repo/$arch
```
